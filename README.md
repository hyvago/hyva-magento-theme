# Hyvä Theme Open Source

## Overview

This repository hosts the open-source version of the Hyvä Theme for Magento 2. Hyvä replaces the default Magento frontend with a lightweight, high-performance solution built with modern technologies.

## Features

* Performance-first architecture: Hyvä uses a minimal frontend stack, delivering significantly fewer assets and faster load times. ([Neklo][1])
* Developer-friendly: Built with Tailwind CSS (utility-first styling) and Alpine.js for interactivity, replacing RequireJS/Knockout from older themes. ([evrig.com][2])
* Reduced complexity: The theme is built from scratch with blank base templates, fewer dependencies, easier maintenance. ([hyva.io][3])
* Improved Core Web Vitals and SEO readiness: Fast FCP/LCP and low CLS out-of-the-box. ([Neklo][4])
* Clean structure and modularity for easier customization and future extensions. ([Meetanshi - Magento & Shopify Agency][5])

## Free Hyvä Templates & Add-ons

Hyvä supports a growing ecosystem of ready-to-use templates and sub-themes that work on top of the core theme:

* Pre-designed page layouts (home, category, product) for faster launches. ([Meetanshi - Magento & Shopify Agency][5])
* UI component library Hyvä UI: a collection of modular, ready-to-use components to plug into your store. ([hyva.io][6])

## Getting Started

1. Clone this repository
2. Install and enable the theme in your Magento 2 setup (ensure compatibility with your version) ([docs.hyva.io][7])
3. Create or install a child theme if you plan to customise styles/templates
4. Run Magento setup commands to upgrade, compile, deploy static content
5. Configure your store to use the Hyvä theme via Content → Design → Configuration

## Why Choose Hyvä?
